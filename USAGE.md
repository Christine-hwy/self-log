# 使用指南

## 🎯 核心功能

### 1. 添加旅行记忆

**方式一：点击地球**
- 直接在 3D 地球上点击任意位置
- 系统会自动填充经纬度坐标
- 填写地点名称、日期和描述
- 上传照片和视频

**方式二：使用按钮**
- 点击左侧面板底部的 "Add New Memory" 按钮
- 手动输入地点信息和坐标
- 上传照片和视频

### 2. 上传媒体文件

**照片上传**
- 支持多张照片同时上传
- 支持的格式：JPG, PNG, GIF, WebP 等
- 照片会显示在记忆详情页的网格中
- 点击照片可以全屏查看

**视频上传**
- 支持多个视频文件
- 支持的格式：MP4, WebM, MOV 等
- 视频会显示为可播放的播放器
- 注意：大文件可能需要较长加载时间

### 3. 查看旅行记忆

**在地球上查看**
- 悬停在标记点上查看简要信息
- 点击标记点查看完整详情（包括所有照片和视频）
- 缩放地球会自动显示/隐藏地点标签

**在侧边栏查看**
- 左侧面板显示所有记忆列表
- 点击列表项可以查看完整详情
- 统计卡片显示总地点数、照片数、视频数

### 4. 地球交互

**基本操作**
- 🖱️ 拖拽：旋转地球
- 🔄 滚轮：缩放视图
- 🔍 放大：显示地址标签（例如：广东 广州）
- ⏸️ 停止交互：地球会自动旋转

**缩放级别说明**
- 默认视图：可以看到整个地球和所有标记点
- 中等缩放：标记点更清晰，开始显示地址标签
- 近距离缩放：清晰显示地点名称和详细标签

## 📊 统计面板

左侧面板顶部显示三个统计卡片：

1. **Places（地点）**
   - 显示你去过的地方总数
   - 紫色渐变卡片

2. **Photos（照片）**
   - 显示上传的照片总数
   - 靛蓝色渐变卡片

3. **Videos（视频）**
   - 显示上传的视频总数
   - 紫罗兰色渐变卡片

## 💾 数据存储

所有数据都存储在本地浏览器中：

- **位置信息**：经纬度、地点名称、日期
- **文本内容**：旅行描述和笔记
- **媒体文件**：照片和视频（转换为 Base64 格式）

**注意**：刷新页面后新添加的数据会丢失。要永久保存数据，需要手动编辑 `src/data/site.ts` 文件。

## 🎨 视觉特色

- **深色主题**：舒适的深色背景，减少眼睛疲劳
- **紫色/靛蓝渐变**：高级的配色方案
- **毛玻璃效果**：现代化的半透明面板
- **流畅动画**：Apple Keynote 级别的过渡效果
- **星空背景**：闪烁的星星增加氛围感

## 🔧 高级功能

### 地址标签显示逻辑

地球上的地址标签会根据缩放级别自动显示/隐藏：

- **远距离**（altitude > 2）：只显示标记点，不显示标签
- **近距离**（altitude < 2）：显示地点名称标签

例如：
- 缩放到中国 → 看到 "Paris, France"、"Tokyo, Japan" 等标签
- 缩放到广东 → 看到 "广州" 标签

### 记忆详情弹窗

点击标记点或列表项会打开详情弹窗，显示：

- 🗺️ 地点名称和日期
- 📝 完整描述
- 📷 照片网格（可点击放大）
- 🎥 视频播放器
- 📍 精确坐标

## ⌨️ 快捷提示

界面底部会显示快捷提示：
- "Zoom in to see location labels" - 提示缩放查看标签
- "Click markers to view details" - 提示点击查看详情

## 🆘 获取帮助

点击右下角的 **?** 帮助按钮查看使用指南。

## 📱 响应式设计

网站完美适配：
- 💻 桌面电脑
- 📱 平板设备
- 📲 手机（侧边栏会折叠到顶部）

## 🎯 最佳实践

1. **照片优化**：建议上传前压缩照片，避免文件过大
2. **视频大小**：视频文件建议不超过 50MB
3. **描述文字**：添加详细的旅行描述，让记忆更生动
4. **定期备份**：将重要数据保存到 `src/data/site.ts`
